# PROMPT DI IMPLEMENTAZIONE

Ecco il prompt per inserire questa sezione nel sito GUIRIMì:

```
Devo aggiungere una sezione "Risorse e Letture Consigliate" alla piattaforma GUIRIMì (Next.js + Tailwind).

POSIZIONE: Può essere implementata in DUE modi (scegli in base all'architettura):

OPZIONE A — Sezione dentro ogni Area
- Aggiungere una tab/sezione "📚 Letture consigliate" alla fine di ogni pagina area (/learn/[areaSlug])
- Visibile DOPO l'elenco dei moduli
- Collassabile per non appesantire la pagina
- Filtro per livello (🟢 Principiante / 🟡 Intermedio / 🔴 Avanzato)

OPZIONE B — Pagina dedicata (/risorse)
- Nuova route /risorse o /letture con navigazione laterale per area
- Sidebar con le 6 aree (stessa sidebar dell'indice contenuti)
- Ogni area mostra i libri divisi per livello
- Possibilità di filtrare per livello globalmente

OPZIONE CONSIGLIATA: B (pagina dedicata) — è più scalabile, non appesantisce le pagine area, e può diventare una risorsa standalone.

STRUTTURA DATI:
Creare un file /content/risorse.ts (o .json) con struttura:

```ts
interface Book {
  title: string;
  author: string;
  publisher: string;
  level: 'principiante' | 'intermedio' | 'avanzato';
  note: string;
  isbn?: string;
  url?: string; // link Amazon/IBS/Feltrinelli (opzionale, con tag affiliazione futura)
  area: string;
  subArea?: string; // es. "diritto-amministrativo" o "diritto-lavoro" per Area 5
}
```

UI COMPONENTI:
1. BookCard — card con titolo, autore, editore, badge livello (🟢🟡🔴), nota breve
2. LevelFilter — toggle/tabs per filtrare per livello
3. AreaResourceSection — sezione per area con libri raggruppati per livello
4. Sezione "Risorse Online Gratuite" in fondo con link a Brocardi, Normattiva, Treccani, ecc.

DESIGN:
- Stile coerente con il resto di GUIRIMì (Tailwind, font-serif per titoli, colori navy/gold)
- Badge livello colorati: verde, giallo/ambra, rosso
- Card con hover effect discreto
- Responsive: 1 colonna mobile, 2 colonne tablet, 3 colonne desktop
- Link esterno con icona ExternalLink e target="_blank"

Usa il file RISORSE-LETTURE-GIURIMI.md come data source per popolare i dati.
Struttura pagina: header con intro → filtri livello → aree con libri → risorse gratuite online
```
