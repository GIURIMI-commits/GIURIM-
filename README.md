## Il Problema

Milioni di italiani cercano di capire il diritto ogni giorno. Trovano manuali da 1000 pagine, siti istituzionali impenetrabili, video YouTube di qualità variabile e forum pieni di disinformazione. Il risultato è sempre lo stesso: **abbandono, confusione, sfiducia**.

Non esiste una risorsa che sia contemporaneamente gratuita, verificata, interattiva, con percorso guidato e accessibile a chi parte da zero.

**GIURIMì colma questo vuoto.**

## Cos'è GIURIMì

GIURIMì è un'academy digitale che trasforma il diritto italiano — dalla Costituzione al sistema giudiziario — in percorsi di apprendimento chiari, strutturati e verificati da professionisti legali.

Non è un manuale. Non è consulenza legale. È la **rampa di lancio** che prepara chiunque a comprendere come funziona la legge in Italia.

<br/>

### Principi Guida

| | Principio | Cosa significa in pratica |
|:---:|:---|:---|
| **◼** | **Accessibilità linguistica** | Ogni concetto spiegato prima in linguaggio quotidiano, poi in linguaggio tecnico |
| **◼** | **Percorso guidato** | Nessuno è lasciato solo — ogni lezione dice da dove vieni e dove vai |
| **◼** | **Verificabilità** | Ogni affermazione collegata alla fonte normativa ufficiale |
| **◼** | **Impatto sociale** | Orientato all'empowerment del cittadino, non al profitto |

<br/>

### Cosa NON è

```
✗  Non è consulenza legale — mai sostituisce un avvocato
✗  Non è un manuale universitario — lo prepara, non lo sostituisce
✗  Non è una quiz app per concorsi — costruisce fondamenta, non simula esami
✗  Non è una banca dati — spiega il contesto, non archivia articoli
```

### Percorsi Personalizzati

L'utente non deve completare tutte le 63 lezioni. All'onboarding sceglie il proprio profilo e riceve un percorso su misura.

| Profilo | Percorso | Lezioni | Tempo |
|:--------|:---------|:-------:|:-----:|
| Liceale / Educazione Civica | Fondamenta + Costituzione + Ord. Giudiziario | ~18 | ~5h |
| Studente 1° anno Giurisprudenza | Fondamenta + Costituzione + Privato + Penale | ~38 | ~10h |
| Cittadino / Autodifesa | Fondamenta + Contratto + PA + Sistema Pratico | ~22 | ~6h |
| Concorsista | Curriculum completo + quiz avanzati | 63+ | ~15h+ |
| Professionista non-legale | Fondamenta + Contratto + Lavoro + Pratica | ~12 | ~3,5h |
| Content Creator | Percorso Cittadino + template + fonti verificate | ~22+ | ~6h+ |

## Tech Stack

| Layer | Tecnologia | Motivazione |
|:------|:-----------|:------------|
| Frontend | **Next.js 14** + TypeScript | SSR, mobile-first, SEO per le lezioni |
| Stile | **Tailwind CSS** | Prototipazione rapida, responsive nativo |
| Auth + DB | **Supabase** (PostgreSQL) | Auth integrato, RLS, free tier per MVP |
| Contenuti | **MDX** (Markdown + JSX) | Lezioni come testo + componenti interattivi |
| Deploy | **Vercel** | CI/CD automatico, zero-config |
| Email | **Resend** | Retention quiz, newsletter aggiornamenti |
| Analytics | **PostHog** | Privacy-first, eventi custom per metriche |

## Quick Start

```bash
# 1. Clona il repository
git clone https://github.com/tuouser/guirimi.git
cd guirimi

# 2. Installa dipendenze
npm install

# 3. Configura le variabili d'ambiente
cp .env.example .env.local
# → Compila con le tue credenziali Supabase

# 4. Setup database
npx supabase db push

# 5. Avvia in sviluppo
npm run dev
```

Apri `http://localhost:3000` — il gioco è fatto.

## Metriche di Impatto (Non Soldi)

GUIRIMì misura il successo con metriche di utilità reale, non di monetizzazione.

| Metrica | Come | Target MVP |
|:--------|:-----|:----------:|
| **Retention di comprensione** | Pre-test → Post-test → Retention test (7gg) | ≥65% retention |
| **Completion rate** | % utenti che completano almeno 1 modulo | ≥40% |
| **Time-to-value** | Tempo da sign-up a primo modulo completato | <3 giorni |
| **Self-efficacy** | Survey 1-10 dopo primo modulo | ≥7/10 |
